Explicación de Markdown

1. Encabezados

Se utilizan #, ##, etc., para crear títulos de diferentes niveles. Por ejemplo:

# Encabezado de nivel 1
## Encabezado de nivel 2
### Encabezado de nivel 3

2. Negrita y cursiva

Para aplicar negrita, usa **texto** y para cursiva, usa *texto*. Para ambos estilos a la vez, usa ***texto***. Ejemplo:

**Este es un texto en negrita**
*Este es un texto en cursiva*
***Este es un texto en negrita y cursiva***

3. Listas

Las listas no ordenadas se crean con - y las ordenadas con 1., 2., etc. Ejemplo de lista no ordenada:

- Elemento 1
- Elemento 2
- Elemento 3

Ejemplo de lista ordenada:

1. Primer paso
2. Segundo paso
3. Tercer paso

4. Enlaces e imágenes

Para insertar enlaces, utiliza la sintaxis [texto](URL). Para imágenes, la sintaxis es ![texto alternativo](URL). Ejemplo de un enlace:

[Visita GitHub](https://github.com)

Ejemplo de una imagen:

![Descripción de la imagen](https://ruta-de-la-imagen.com/imagen.jpg)

5. Bloques de código

Los bloques de código se indican con tres acentos graves (```) y se puede especificar el lenguaje para resaltar la sintaxis. Ejemplo de bloque de código en PHP:

```php

```

6. Citas

Las citas se crean utilizando el símbolo >. Ejemplo:

> "La vida es lo que pasa mientras estás ocupado haciendo otros planes." – John Lennon
"La vida es lo que pasa mientras estás ocupado haciendo otros planes." – John Lennon

7. Tablas

Se usan | para crear tablas con columnas y filas. Ejemplo:

| Hobbies       | Horas por semana |
| ------------- | ---------------- |
| Leer          | 5 horas          |
| Programar     | 10 horas         |
| Deporte       | 3 horas          |
Hobbies Horas por semana
Leer 5 horas
Programar 10 horas
Deporte 3 horas

8. Tareas

Las listas de tareas utilizan - [ ] para tareas pendientes y - [x] para tareas completadas. Ejemplo:

- [ ] Estudiar para el examen
- [ ] Completar el ejercicio de Markdown
- [x] Entregar la tarea en GitHub
  • Estudiar para el examen
  • Completar el ejercicio de Markdown
  • Entregar la tarea en GitHub
